# Vibe Master

Sistema de tabletop focado no mestre (DM), para uso local com compartilhamento de tela no Discord.

## Objetivos

- [ ] Tabuleiro interativo com peças movíveis
- [ ] Sistema de rolar dados
- [ ] Gerenciamento de NPCs e monstros
- [ ] Persistência de aventuras no banco de dados
- [ ] Biblioteca de módulos e personagens
- [ ] Interface otimizada para compartilhamento de tela

## Stack

- **Frontend/Backend:** Next.js 14+ (App Router)
- **Database:** PostgreSQL com Drizzle ORM
- **Styling:** Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

## Banco de Dados

```bash
npm run db:push   # Aplicar schema
npm run db:studio # Abrir Drizzle Studio
```
